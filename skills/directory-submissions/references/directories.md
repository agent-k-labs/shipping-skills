# Directory database

Prioritized list of directories for vibe-coded apps. Tiers are ordered by expected return (traffic quality × backlink value ÷ effort). Submit Tier 1 on day 1, Tier 2 in week 1, Tier 3 in the first month.

## Tier 1 — Submit day 1 (highest traffic, best quality)

| Directory | URL | Best for | Cost | Notes |
|-----------|-----|---------|------|-------|
| Product Hunt | producthunt.com | All SaaS/apps | Free | Schedule a real launch day; don't just add to the site. See the product-hunt-launch skill. |
| Indie Hackers | indiehackers.com/products | Solo founder tools | Free | Add product page + intro post in community |
| Show HN | news.ycombinator.com | Dev/technical tools | Free | Text submission linking to your site; be ready to answer technical questions |
| Uneed | uneed.best | All tools | Free tier | Launch-calendar model like PH; less competitive |

## Tier 4 — If relevant to your niche

- **Built-with showcases**: "Made with Bolt", "Built on Lovable", Replit gallery, v0 showcase — high-affinity audience if you built with that tool
- **Marketplace listings**: Chrome Web Store (extensions), Notion/Figma/Linear integration galleries, Slack App Directory
- **Agent/MCP ecosystem** (if your product is agent-adjacent): skills.sh, MCP registries, awesome-* GitHub lists
- **Niche searches**: `[industry] tools directory`, `[audience type] resources`, `best [category] newsletter` (newsletters often have tool roundups)

## Freshness note

Directory landscapes shift fast — sites die, pricing changes, new ones appear. If a listed URL 404s, skip it and note it. When submitting for an AI product, also search "[current year] AI tool directories" for newcomers with launch-boost mechanics.
